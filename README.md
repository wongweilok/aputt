# APU Terminal Timetable (aputt)

## About

A small terminal application written in GO that display Asia Pacific University timetable.

## Prerequisites

* Go version 1.16.2 or above. Refer [this](https://golang.org/doc/install) page on how to install Go.

## Installation

```
$ go get github.com/wongweilok/aputt
```

**\*NOTE**: If your virus scanner (especially Windows Defender) detect virus from this program, it's false positive.

For more information, please refer the [Golang Documentation](https://golang.org/doc/faq#virus).
